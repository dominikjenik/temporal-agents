# Pending Lessons

Staging area for lessons captured during workflow execution.
Review regularly and promote valuable insights to agent definitions.

## Format

Each lesson entry:
- **timestamp**: ISO 8601 (UTC)
- **workflow_id**: Temporal workflow ID
- **agent_type**: e.g. developer, tester, devops-zbornik
- **outcome**: success | failure
- **lesson_text**: free-form text describing what was learned
